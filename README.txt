Installasi :
1. Install Arduino IDE
2. Install Aplikasi Loket dan Laboratorium
3. Hubungkan alat dengan access point/ Wifi
4. Persiapkan database utama
5. Konfigurasi ID dan Lokasi Barang sesuai kebutuhan
6. Alat siap digunakan

Penggunaan Alat:
1. Scanner Loket
- Tempelkan kartu RFID
- Masukan data melalui aplikasi loket
- Kartu RFID siap digunakan
2. Scanner Lab
- Dekatkan scanner dengan kartu RFID
- Tekan tombol pengecekan data
- Rubah status pengujian sesuai kebutuhan
- Tekan tombol send untuk mengirimkan update data ke database
3. Scanner Reset
- Tempelkan kartu RFID pada scanner
- Tekan tombol Reset
- Secara otomatis data pada RFID Card dan Database ter-reset
